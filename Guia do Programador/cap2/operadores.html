Operadores em JavaScript

Em JavaScript, os operadores se dividem em algumas categorias.

  • Aritméticos ou matemáticos
    • Adição
    • Subtração
    • Multiplicação
    • Divisão
    • Módulo ou resto da divisão
    • Incremento
    • Decremento
  • Relacionais ou de comparação
  • Lógicos
  • De strings
  • De bits
  • Atribuição
  • Especiais

Adição

O operador de adição pode ser usado para somar dois números ou concatenar duas strings.

  • Se os dois operandos forem números, ele fará uma soma.

    • 8 + 3 = 11

    9.7 + 4 = 13.7

  • Se os dois operandos forem strings, ele fará uma concatenação.

    • "Olá " + "Mundo!" = "Olá Mundo!"

    "12" + "12" = "1212"

  • Se apenas um dos valores for string, o interpretador irá converter para uma string.

    • "Olá" + 23.05 = "Olá23.05"

    "função" + function(){} = "funçãofunction(){}"
