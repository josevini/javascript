Adição

O operador de adição pode ser usado para somar dois números ou concatenar duas strings.

  • Se os dois operandos forem números, ele fará uma soma.

    • 8 + 3 = 11

    9.7 + 4 = 13.7

  • Se os dois operandos forem strings, ele fará uma concatenação.

    • "Olá " + "Mundo!" = "Olá Mundo!"

    "12" + "12" = "1212"

  • Se apenas um dos valores for string, o interpretador irá converter os dois para string.

    • "Olá" + 23.05 = "Olá23.05"

    "função" + function(){} = "funçãofunction(){}"

Subtração

O operador de subtração calcula a diferença entre dois operandos numéricos.

  • Se dois operandos são valores numéricos, o resultado será a diferença.

    • 12 - 4 = 8

    5.5 - 3 = 2.5

  • Se os dois operandos são strings, o resultado será NaN.

    • "Java" - "Script" = NaN (Not a Number)

  • Se os dois operandos são strings numéricas, o interpretador fará a conversão.

    • "10" - "8" = 2

Multiplicação

Este operador multiplica dois operandos numéricos.

  • Se os dois operandos forem numéricos, o resultado será a multiplicação dos dois.

    • 4 * 2 = 8

Divisão

Este operador realiza uma divisão entre o primeiro operando e o segundo operando.

  • Se os dois operandos forem strings, o resultado será NaN.

    • "Olá" / "Mundo" = NaN

  • Se os dois operandos forem strings numéricas, o resultado será uma divisão.

    • "4" / "2" = 2
