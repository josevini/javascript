Expressões Regulares

O que é?

As RegEx ou Expressões Regulares são conjuntos de símbolos e caracteres literais que têm como funcionalidade validar se uma string casa com os padrões fornecidos na expressão.

Construtor

As regex têm uma função construtora chamada RegExp() que recebe dois parâmetros: pattern e flags

Sintaxe: RegExp(pattern[, flags])

Literal

Há também a possibilidade de utilizar a sintaxe literal.

Sintaxe literal: var exp = \pattern\flags

Argumentos e definições

Pattern ou Padrão: é o argumento onde definimos a expressão que será utilizada para validar os dados.

Flags ou Modificadores: este argumento define alguns requisitos para a busca. Um deles é diferenciar ou não letras maiúsculas e minúsculas.

Modificadores

Os Modificadores impõem determinadas condições às buscas.

ModificadorCondição
iA busca não faz distinção entre letras maiúsculas e minúsculas.
gA busca será para todas as ocorrências e não somente para a primeira ocorrência encontrada.
mA busca será multilinhas.
giTodas as condições serão atendidas.

Agrupadores

Os Agrupadores se destinam a definir uma faixa de busca.

AgrupadorDescrição
[xyz]A busca se faz para os caracteres x, y, z.
[^xyz]A busca se faz para caracteres diferentes de x, y, z.
[0-9]A busca se faz para dígitos de 0 até 9.
[a-z]A busca se faz para letras minúsculas de a até z.
[A-Z]A busca se faz para letras maiúsculas de A até Z.
[a-zA-Z]A busca se faz para letras minúsculas e maiúsculas de a até Z.
[x|y|z]A busca se faz para x ou y ou z.

Metacaracteres

Os Metacaracteres são símbolos especiais, com significado próprio, para inserção em um padrão de busca.

MetacaractereDescrição
. (ponto)Busca qualquer caractere, menos os de nova linha e fim de linha
\wBusca qualquer caractere compreendido em [a-Z] [0-9] e underscore (_).
\WBusca qualquer caractere não compreendido em [a-Z] [0-9] e underscore (_).
\dBusca qualquer caractere
\DA busca se faz para letras maiúsculas de A até Z.
\sA busca se faz para letras minúsculas e maiúsculas de a até Z.
\SA busca se faz para x ou y ou z.
