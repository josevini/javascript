Objeto Date

O que é?

O Objeto Date é um construtor de datas e horários. Existem 4 sintaxes para criar um objeto date.

var d = new Date() var d = new Date(milissegundos) var d = new Date(data_string) var d = new Date(ano, mes, dia, hora, minuto, segundo, milissegundo)

new Date()

Ao usar o construtor Date sem passar argumentos para ele, cria-se um objeto contendo a data e hora atual.

var agora = new Date() alert(agora) // Sat Mar 27 2021 10:07:14 GMT-0300 (Hora padrão de Brasília)

new Date(milissegundos)

Passar o argumento milissegundos ao construtor Date resulta na criação de um objeto contendo a data e hora equivalentes ao valor passado pelo construtor.

var agora = new Date(284782462) alert(agora) // Sun Jan 04 1970 04:06:22 GMT-0300 (Hora padrão de Brasília)

new Date(data_string)

Outra opção é passar o conjunto data-hora como argumento para o construtor.

  • 'MM-dd-aaaa hh:min:seg'
  • 'aaaa/MM/dd hh:min:seg'
  • 'MM/dd/aaaa hh:min:seg'
